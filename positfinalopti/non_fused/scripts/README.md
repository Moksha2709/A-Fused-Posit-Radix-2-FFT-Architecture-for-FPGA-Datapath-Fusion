# Non-Fused Posit FFT Scripts

This directory houses the helper scripts for the Non-Fused Posit FFT baseline architecture. It includes Python scripts to dynamically generate parameterized testbenches and verify simulated RTL results against double-precision NumPy and MATLAB reference models. Additionally, a Vivado Tcl script is included to automate the FPGA synthesis process and capture resource estimation metrics for comparing the design against the fused implementation.
