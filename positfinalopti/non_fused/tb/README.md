# Non-Fused Posit FFT Testbench

This directory contains the Verilog HDL testbench file (`tb_fft16.v`) for the baseline Non-Fused Posit FFT design. The testbench functions to initialize the test environment, drive the design inputs with realistic test vectors, and log the resulting outputs from the top-level FFT pipeline. This logs the output data required to evaluate the precision characteristics and correctness of the standard modular Posit arithmetic implementation.
