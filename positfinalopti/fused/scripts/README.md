# Fused Posit FFT Scripts

This directory houses the Python and Vivado Tcl scripts used to manage the development and analysis of the Fused Posit FFT. It includes a Python script to dynamically generate the parameterized testbench for different Posit configurations and another script to compare simulated outputs against a MATLAB/NumPy golden reference. A Vivado Tcl script is also provided to automate the RTL synthesis and resource utilization runs on target FPGA devices.
